# Information

* This Plugin Allow You To Use Console From MCPE.
* Issue Console Commands Directly From Game Console.
* This Plugin Will Not Allow Non-Ops To Use Console Without Permission. 

# Commands

* /console <args..>
* /c <args..>

# Permissions

* `mcpetodiscord.cmd`
